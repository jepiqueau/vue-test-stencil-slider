# vue-test-slider
Sample project that shows an integration of [jeep-slider web component build with StencilJS](https://github.com/jepiqueau/stencil-slider) 
in Vue.js App.

## Project setup
To start a new project using Vue.js, clone this repo to a new directory:

```bash
git clone https://github.com/jepiqueau/vue-test-stencil-slider.git my-app
cd my-app
git remote rm origin
```


```
npm install
```
